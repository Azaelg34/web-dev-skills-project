WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the Content of a web page. Examples of HTML include headings, paragraphs, images, links, lists, etc.

Check out the on HTML Tutorial on w3schools.com

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the Style of a web page. Examples of CSS include font styles and colour, background colour, borders, dimensions, layout, etc.

Check out the CSS tutorial on w3schools.com

Learn JavaScript

javascript is the programming langauge of the web. Javascript helps to make web pages interactive and dynamic. Some common JavaScript tasks are:

  • add new HTML to the page,
  • Change the existing content/style,
  • React to user events (mouse clicks, key presses, etc.)

Check out the Javascript Tutorial on javascript info.com
